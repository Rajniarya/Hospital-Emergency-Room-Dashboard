# Hospital Emergency Room Dashboard

An interactive Excel dashboard analyzing Hospital Emergency Room data.

## 📊 Project Overview
- Analyzed patient data for years 2023 and 2024
- Built an interactive dashboard with slicers for Year and Month filtering

## 🔍 Key Insights
- Total Patients: 3845
- Average Wait Time: 35.04 minutes
- Satisfaction Score: 5.06
- 59% patients attended On-time, 41% delayed
- Equal admission split: 50% Admitted, 50% Not Admitted

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Slicers
- Charts (Pie, Bar, Donut)
